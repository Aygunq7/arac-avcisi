# Araç Avcısı v28

Bu sürümde filtre motoru kesin hale getirildi.

- Fiyat filtresi girildiyse fiyatı okunamayan veya aralık dışında kalan ilan listeye girmez.
- Yıl filtresi girildiyse yılı okunamayan veya aralık dışında kalan ilan listeye girmez.
- KM filtresi girildiyse km bilgisi okunamayan veya limit üstü olan ilan listeye girmez.
- Otomatik vites seçildiyse manuel veya vites bilgisi okunamayan ilan listeye girmez.
- Şehir seçildiyse şehir bilgisi okunamayan ilan listeye girmez.
- Letgo/Facebook otomatik listeye alınmaz, sadece siteyi aç butonu kullanılır.

Sahibinden gibi otomatik erişimi sınırlayan kaynaklarda sonuç az/boş gelebilir. Bu sürümün hedefi çok sonuç değil, doğru filtrelenmiş gerçek sonuçtur.
